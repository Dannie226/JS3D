Vector3

new JS3D.Vector3(x,y,z): {three: three.Vector3, cannon: CANNON.Vec3}
x - x value of the Vector
y - y value of the Vector (up)
z - z value of the Vector

World

new JS3D.World(target, gravity): this
target: HTMLDOMElement
gravity: JS3D.Vector3()

Returned Items

scene: THREE.Scene
cameras: [THREE.PerspectiveCamera]
renderer: THREE.WebGLRenderer
cWorld: CANNON.World
bodies: Array of all bodies added to the World
add(body: JS3D.ShapeBody): null
update():null
addCamera(camera: THREE.Camera): null, adds a camera to the scene
setVeiwMode(mode: Number):null, sets which camera to use for rendering

Box Body

new JS3D.BoxBody(geometry,material,mass,position): {mesh: THREE.Mesh,physicsBody:CANNON.Body}
geometry: THREE.BoxGeometry or THREE.BoxBufferGeometry
material: THREE.MeshBasicMaterial
Mass: Number
Position: JS3D.Vector3();

new JS3D.SphereBody(geometry,material,mass,position): {mesh: THREE.Mesh,physicsBody:CANNON.Body}

Character

new JS3D.Character(width, height, depth, material, position, model): {mesh: THREE.Mesh,physicsBody:CANNON.Body}
width: Number, width of the hitbox
height:Number, height of the hitbox
depth: Number, depth of the hitbox
material: THREE.MeshBasicMaterial
position: JS3D.Vector3
model: optional THREE.Mesh

Character Controller

camera: THREE.Camera
moveSpeed: Number
fixedCam: Boolean

Returned Items

left():moves character left with respect to the camera, and sometimes the camera based on whether you have fixed camera on
right()
forward()
back()
jump(jumpVelocity: Number): null
